Edit `_config.yml` file in the `Theme` folder to set google adsense.

Here are 2 Ads Units for default (Auto Size), which is `footer` and `banner`. 

Pub Id and Ads Slot Id can be found in your Google Adsense Settings.
  
```
google_adsense:
  enabled: true
  auto_ad: true
  pub_id: 
  footer_slot: 
  banner_slot: 
```
