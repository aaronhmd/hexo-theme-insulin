## Tags page
>Add a tags page contains all tags in your site.

**1. Create a page named tags**
```
$ hexo new page "tags"
```
**2. Edit tags page**

Set page type to tags
```
title: All Tags
date: 2020-02-02 20:00:02
type: "tags"
```
